## 1. SQLens 是什么

SQLens 是一个**图形化的网站安全检测工具**。它能自动检测你的网站是否存在 **SQL 注入漏洞**——这是最常见的网站安全漏洞之一。

**打个比方：**
- 你的网站就像一栋房子
- SQL 注入就像小偷找到了没锁的后门
- SQLens 就是帮你检查所有门窗是否关好的工具

### 它能做什么？

| 功能 | 小白说法 |
|------|---------|
| 🔍 漏洞扫描 | 自动检测网站有没有安全问题 |
| 🤖 AI 助手 | 像 ChatGPT 一样和你对话，帮你分析 |
| 📊 生成报告 | 把扫描结果整理成文档 |
| 📥 导入请求 | 从 Burp Suite、cURL 等工具导入数据 |

## 4. 扫描一个网站

### 场景：检测你的网站是否有 SQL 注入

#### 方法一：一键全自动（推荐 ✅）

```
步骤 1: 复制你的网站 URL
        比如 http://example.com/page?id=1

步骤 2: 粘贴到输入框
        按 Ctrl+V 或右键粘贴

步骤 3: 点击「✨ 一键全自动」
        或者按 Ctrl+Enter (⌘+⏎)

步骤 4: 等待扫描完成
        喝杯咖啡 ☕，扫描会自动进行

步骤 5: 查看结果
        右侧日志区会显示实时进度
        底部结果区会展示发现的数据
```

#### 方法二：手动配置

```
步骤 1: 粘贴 URL
步骤 2: 在左侧「目标」面板调整参数
步骤 3: 选择扫描模板（如「快速检测」）
步骤 4: 点击「一键全自动」
```

### 导入 Burp Suite 请求

```
步骤 1: 在 Burp Suite 中右键请求 → Copy to file
步骤 2: 在 SQLens 中点击「📥 导入」
步骤 3: 选择「Burp Suite」选项卡
步骤 4: 粘贴内容 → 点击「解析并应用」
```

## 7. 常见问题

### Q: 启动后报 "sqlmapapi 未连接"？

**原因：** sqlmap 没有自动启动。

**解决方法：**
```bash
# 在终端运行
cd SQLens
npm run setup    # 下载 sqlmap
npm run dev      # 重新启动
```

### Q: 扫描进度很慢？

- 可以尝试在「优化」面板调高「线程数」（默认 1，可以调到 5）
- 选择「快速检测」模板
- 如果网速慢，可以增加请求延时

### Q: AI 说 "未配置"？

1. 点击左侧「⚙️ 设置」
2. 输入 DeepSeek API Key
3. 点击「测试连接」
4. 保存

免费获取 API Key：https://platform.deepseek.com

### Q: 如何切换暗色模式？

- 点击右上角的 🌙/☀️ 按钮
- 或按快捷键 `Ctrl+Shift+T`

### Q: 扫描结果怎么导出？

- **数据导出：** 在数据库浏览器中点击 CSV/JSON 按钮
- **报告导出：** 点击左侧「📊 报告」→「生成报告」→「导出 HTML/Markdown」

### Q: 如何批量扫描多个网站？

1. 点击「📋 批量」按钮
2. 粘贴 URL 列表（每行一个）
3. 点击「开始扫描」
4. SQLens 会自动排队扫描

### Q: 支持哪些快捷键？

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 一键全自动 |
| `Ctrl+Shift+T` | 切换暗色/亮色主题 |
| `Escape` | 关闭弹窗 |

## 💡 使用小贴士

1. **先测自己的网站** — 不要对别人的网站使用，除非有授权
2. **从简单开始** — 先用「快速检测」模板，再慢慢加深
3. **AI 是你的助手** — 遇到不懂的就问 AI
4. **定期扫描** — 每次更新网站后都扫一次
5. **及时修复** — 发现漏洞后按照报告里的修复建议处理
